# Data Extractor
A Python script for extracting data (figures, texts, tables, equations, hyperlinks, etc.) from a research paper PDF or Latex 

### Extracted Elements
- Title
- Abstract
- Paper Metadata
    - Authors
    - Affiliations
    - Emails
    - Publication
    - DOI
    - Date
    - License
- Format Metadata
    - Section Hierarchies
    - Text Metadata
        - Color
        - Style
- Sections
    - Heading
    - Body texts
        - Cross-References
        - Citations
    - Figures
        - Image
        - Caption
        - Labels
    - Tables
        - Caption
    - Links
    - Equations
- References 
- Appendices
